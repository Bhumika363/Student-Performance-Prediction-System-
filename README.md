# Student Performance Prediction System

A professional data-driven application designed to forecast academic outcomes by leveraging **Machine Learning (Linear Regression)** and **Java** backend technologies. The system predicts a student's future scores based on key performance indicators and maintains a persistent history of these records.

## 🚀 Overview
Predicting student success allows for early intervention and personalized academic support. This project provides a bridge between raw academic data and actionable insights, using a regression model to estimate final results based on internal assessments.

## 🛠️ Tech Stack
* **Language:** Java
* **Machine Learning:** Linear Regression
* **Database:** MySQL
* **Backend Connectivity:** JDBC
* **Tools:** Visual Studio Code

## 📊 Prediction Logic
The system calculates the predicted results based on three specific inputs:
1. **Attendance:** Percentage of classes attended.
2. **MST1 Marks:** Score from the first Mid-Semester Test.
3. **MST2 Marks:** Score from the second Mid-Semester Test.

The linear regression model analyzes these variables to output a predicted future score, which is then automatically saved to the MySQL database for record-keeping and tracking.

## 📁 Project Structure
* **`src`**: Contains the core Java logic and database connection files.
* **`model`**: Contains the Linear Regression model implementation.
* **`lib`**: External JAR files and dependencies.
* **`design`**: UI/UX assets and system design documentation.
* **`config`**: Configuration files for environment and database settings.
